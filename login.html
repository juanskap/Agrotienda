Un solo ingreso para clientes y administradores

El mismo acceso servira para compras, seguimiento de pedidos y gestion interna. El sistema definira el destino segun el rol real de cada cuenta.

Cliente

Consulta pedidos, edita perfil, guarda productos y finaliza compras.

Administrador

Gestiona inventario, usuarios, pedidos y reportes en una zona privada.

Iniciar sesion

Ingresa con tu correo y contrasena. Cuando conectemos backend, el sistema identificara si debes entrar a tu cuenta o al panel administrativo.

No hace falta escoger tipo de usuario aqui. El rol se validara desde la base de datos para evitar accesos manuales o inconsistentes.
Importante

La privacidad del administrador no se puede asegurar solo con HTML. Luego agregaremos validacion del lado del servidor, sesiones y permisos por rol.
