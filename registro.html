Crear cuenta de cliente

Este registro esta pensado para compradores. La cuenta de administrador no debe crearse desde la parte publica, sino desde un proceso interno y protegido.

Las cuentas de administrador deben manejarse con permisos especiales. Cuando integremos la base de datos, este rol se controlara por backend y no desde el formulario publico.
